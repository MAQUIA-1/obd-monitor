# OBD Monitor

Personal browser-based OBD dashboard for my vehicle.

## Features

- Web Bluetooth OBD connection
- Speed, RPM, battery current, steering angle
- Fuel, SOC, coolant temperature, HV voltage, 12V voltage, outside temperature
- Demo mode for layout checks

This is tuned for my own OBD/BLE adapter and PID set, so it is not intended as a general-purpose OBD app.

## Run

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
